Politique de confidentialité

Dernière mise à jour : avril 2026

1. Responsable du traitement

Tristan Peyrotty — citruscodex@gmail.com

2. Données collectées

  • Email (inscription, identification)
  • Mot de passe (haché bcrypt 12 rounds — jamais stocké en clair)
  • Données de collection (plantes, événements, photos)
  • Localisation approximative (météo, observatoire — si autorisée par l'utilisateur)
  • Données de navigation (pages consultées, type de navigateur)
  • Données synchronisées entre appareils via nos serveurs sécurisés (hébergés en France, Scaleway SAS, Paris)

3. Finalité du traitement

  • Authentification et gestion du compte
  • Synchronisation des données entre appareils
  • Fonctionnalités météo et observatoire
  • Notifications phénologiques
  • Amélioration du service

4. Base légale

  • Consentement (inscription volontaire)
  • Exécution du contrat (fourniture du service)
  • Intérêt légitime (sécurité, maintenance)

5. Durée de conservation

  • Données de compte : jusqu'à suppression du compte
  • Logs serveur : 90 jours
  • Backups : 30 jours (rotation automatique)

6. Sous-traitants

  • Scaleway SAS — hébergement, emails transactionnels (France)
  • GitHub — déploiement CI/CD (Microsoft Corp., USA — données de code uniquement)

7. Vos droits (RGPD)

Vous disposez des droits suivants :

  • Accès : télécharger toutes vos données (Réglages → Télécharger mes données)
  • Rectification : modifier vos informations depuis les Réglages
  • Effacement : supprimer votre compte et toutes les données associées (Réglages → Supprimer mon compte)
  • Portabilité : exporter vos données en JSON
  • Opposition : vous désinscrire à tout moment

Contact : citruscodex@gmail.com

8. Transferts hors UE

Aucun transfert de données personnelles hors de l'Union Européenne. Les serveurs sont situés en France (Scaleway, Paris).

9. Sécurité

  • Chiffrement TLS 1.3 (HTTPS)
  • Mots de passe hachés bcrypt (12 rounds)
  • Données synchronisées chiffrées AES-256-GCM
  • Backups automatiques avec rotation 30 jours

10. Cookies et stockage local

CitrusCodex n'utilise pas de cookies. L'authentification repose sur un jeton JWT stocké en localStorage (côté navigateur), jamais transmis à des tiers.

Vos données de collection sont synchronisées uniquement vers les serveurs CitrusCodex (Scaleway, France) — aucun transfert vers des prestataires tiers.
